# College-Website-Assignment-
college website assignment 
UwAmp will need to be installed, once it has been installed the SQL or XML filed provided with the assignment will need to be imported to the database.

After importing, copy the code files on here into the UwAmp folders "www" file as it will not work otherwise 

After importing and moving the files to their respective areas, the website can be accessed via opening the UwAmp exe file and clicking on "www" there it will say "college" which the user will tap and be directed to the webpage for it then to be viewed.
